## 🛠️ STACK TECNOLÓGICO RECOMENDADO

```
Frontend:
- Next.js 15 + React 19
- TypeScript 5.3
- Tailwind CSS 3.4
- shadcn/ui
- Zustand (state management)

Backend:
- Next.js API Routes
- Node.js 20
- Prisma ORM
- Zod (validación)

Database:
- PostgreSQL (Neon o RDS)
- Vector DB: Pinecone/Weaviate/pgvector

AI/LLM:
- OpenAI (GPT-4, embeddings)
- Anthropic (Claude)
- Google Gemini
- LangChain (orquestación)

Auth:
- NextAuth.js
- OAuth2

Deployment:
- Vercel (frontend)
- Cloud Run/Lambda (backend)
- PostgreSQL managed (DB)

Monitoring:
- Sentry (errors)
- LogRocket (sessions)
```
